# Internal Dashboard Portal

## Overview
This project is a **frontend-only dashboard application** built with **React and TypeScript (TSX)**.  
It serves as a centralized internal portal that provides quick access to multiple company tools and resources through a card-based interface.

Each card represents a service or resource and contains embedded navigation links, making it easy for staff to access internal systems from a single location.

The application contains **no backend logic** and is deployed as a **static web application**.

---

## Features
- Card-based dashboard UI
- Embedded navigation links for internal tools
- Built with React + TypeScript
- Fast, lightweight, and easy to maintain
- No backend or database dependency
- Optimized for internal company use

---

## Tech Stack
- **Frontend:** React, TypeScript (TSX)
- **Build Tool:** npm
- **Server:** Linux
- **Web Server / Reverse Proxy:** NGINX
